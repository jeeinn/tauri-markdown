# tauri-markdown
A simple local markdown tool, use Tauri &amp; editor &amp; vue

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```
### Another Terminal

```
npm run tauri dev
```

### Compiles and minifies for production

```
npm run prod
```

### Lints and fixes files

```
npm run lint
```
